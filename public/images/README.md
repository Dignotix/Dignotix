# Image Assets

Place all required images in this directory. See the main README.md for the complete list of required images and their descriptions.

## Quick Reference

Required images:
- `hero_logo_with_tagline.png` - Company logo + tagline
- `hero_device_render.png` - Main product device photo
- `problem_infographic.png` - Problem statement visual
- `solution_icons.png` - Smart/Wearable/Portable icons
- `product_metrics_stats.png` - Product metrics
- `tech_stack_diagram.png` - Technology diagram
- `market_chart.png` - TAM/SAM/SOM visual
- `competitive_table.png` - Comparison table
- `partner_wlb_logo.png` - Partner logo
- `business_model_diagram.png` - Revenue model
- `team_mohammed_abdo.png` - CEO portrait
- `team_mostafa_nasser.png` - CBO portrait
- `milestones_timeline.png` - Roadmap timeline
- `contact_card.png` - Contact information

Run `npm run export:images` to validate all images are present.
